Edit a Topic

Make revisions to an existing topic.

For significant revisions, Help must be generated again (see Generate Output for DITA Map). For minor edits, such as fixing a typo, the HTML page for the topic may be revised to avoid this process. Be sure to make the revision in the XML file as well for future releases.

  1. Open the DITA Bookmap for the product. Go to Resource Manager > File > Open Map. A dialog box displays to navigate to the desired .ditamap file.
  2. From the DITA Bookmap, locate and double-click the desired topic. The View container displays the topic file.
  3. Make desired revisions and click the Save icon of the tool bar. The Save icon becomes disabled.
  4. Perform any additional revisions as desired and Generate Output for DITA Map to verify you didn't break anything.
