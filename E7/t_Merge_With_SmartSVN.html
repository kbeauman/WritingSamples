Merge With SmartSVN

Alternate method of merging a branch folder to the trunk.

  1. Commit, Update, and Tag as described in Steps 1-3 of Merge a Branch to the Trunk.
  2. In the Directories container of SmartSVN, open the working copy (hard drive) of the trunk folder. If necessary, go to Project > Open Working Copy. The Files container populates.
  3. Go to Modify > Merge. The Merge dialog box displays.
  4. Set the following options for the Basic tab:
    1. Merge From - Branch
    2. Type the branch file name in the corresponding text field or click Select to click a folder from the Tag Browser dialog box. The SVN directory for the branch folder displays in Location.
    3. Select Revision Range and type the desired revisions to merge or click Select to choose the range from the Revisions for branch dialog box. Preview and Merge are enabled.
    4. It is recommended to click Preview to ensure all settings are correct, then click Merge.
    A dialog box displays the process and OK is enabled when the merge is complete.
  5. Verify revisions exist in the local trunk folder and select SVN Commit from the context menu.
  6. If necessary, rename repository branch version folder, select Switch from the context menu of the local branch folder, and revise the folder name.
