Publish Release Notes

Publish Release Notes to display on the Knowledgebase.

  1. Log in to KBPublisher:
    • Chrysler - https://workbench-kb.dcctools.com/kb/admin/login.php
    • Fiat - https://fiat.dcctools.com/kb/admin/login.php
    The Home page displays.
  2. Click the toolbar item Articles. By default, the Articles tab displays with a list of Release Notes.
  3. Select the corresponding check box for this version.
    Note: The Status must be Draft, which is indicated by a gray icon as well as a mouse-over tool tip.
  4. Select Set Status from the Actions with selected menu. A second menu displays with Published selected by default.
  5. Click OK.
  6. Click Logout to complete this procedure.
