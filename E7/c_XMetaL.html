Editing Help Text With XMetaL

Use XMetaL to edit Help text and compile Help file.

Tip: Read XMetaL Help.
The XMetaL interface consists of the following three containers:
  • Resource Manager - Displays the DITA Map, which is the table of contents. The File menu is used to compile the Help HTML file. See Generate Output for DITA Map for instructions.
  • View - Displays topic text. Tags On view is recommended. If necessary, go to View > Tags On. Use Plain Text in extreme situations only.
  • Element List - Displays element tags that are in compliance with DITA standards, i.e., if a tag displays you may use it.

Optionally, display the Attribute Inspector for setting element attributes, such as changing a Note to a Tip.
