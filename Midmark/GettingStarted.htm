Getting Started

When you log into the Midmark RTLS Cloud App, the Location Selection page opens with tiles for each top-level location you have access to. Operations users land on the Operations page. Click a tile to open the Facility Dashboard. You can also use the left navigation with the following options:

  • Dashboard - A collection of tiles linked to specific Asset List filters. See Dashboard for additional information.
  • Assets - Opens the Asset List page with assets having an asset tag and has real-time location information. This page is intended to be used to identify and facilitate preventive maintenance. From this page you can open the Map Page, which is a scaled illustration of each floor in a building.
  • Administration - Create, edit, or delete a single asset record, and search the database for an existing asset record. Operations can import multiple assets using the Asset Import page. Click to expand and access the following options:
    • Assets - A complete list of assets. See Asset List for additional information.
    • Users - Add access and assign roles to each user. See User Administration for additional information.
    • Hardware - View network devices and monitor performance. See Hardware for additional information.
    • Locations - Manage the campus, building, floor and room name for each location. See Locations for additional information.
    • Import Venues - Import floor plans for each location.
    • Badge Traffic - View badge and sensor performance. See Badge Traffic for additional information.

Top Bar

The top bar of the browser frame offers the following features, and is available on each Midmark RTLS Cloud App page:

  • Home  - Opens the home page for the location.
  • Location Hierarchy - A breadcrumb with links to parent locations, depending on your organization's structure. For example, Group Name > Campus > Building, or Building > Floor > Room. 
  • Location Menu - A list of locations associated with the tenant.
  • User Profile  - Opens a menu where you can log out of the session.

Filters

Some pages contain a filtering feature that allows you to enter specific criterion that limits the data, such as a specific manufacturer or assets indicating low battery. Click the filter icon  to open the Filter drawer and enter keywords, dates, or other available options. Click Save Filter to add the criteria and column sort order to the Saved Filters menu. Open the Saved Filters menu to apply the filter again.

You can apply multiple filters to the data. You can also Search within your filter results. For example, you want to find an asset but only know the Asset Type and Owner. You can filter on these two data points to see all assets of the type and owner you enter. Your saved filters are stored under your profile and can be accessed on any device.

Watchlists

Another option is to build your own watchlist by selecting the icon  of the asset. You can filter on these selected assets.
